## 🔍 アップロード確認チェックリスト

### GitHubリポジトリで確認すべきこと

#### ✅ ファイルが正しくアップロードされているか
1. `index.html` が存在する
2. `README.md` が最初に表示される
3. すべての `.md` ファイルが存在する
4. `google-apps-script.js` が存在する

#### ✅ GitHub Pagesが有効になっているか
1. Settings → Pages に移動
2. Branch が「main」になっている
3. Folder が「/ (root)」になっている
4. 緑色のボックスにURLが表示されている

#### ✅ 公開URLにアクセスできるか
1. 表示されたURLにアクセス
2. システムのログイン画面が表示される
3. 講師でログインできる
4. レッスン一覧が表示される

## 💡 Tips

### ファイルを一度に選択する方法

#### Windowsの場合
- **Ctrl + クリック**: 個別に選択
- **Shift + クリック**: 範囲選択
- **Ctrl + A**: すべて選択

#### Macの場合
- **Command + クリック**: 個別に選択
- **Shift + クリック**: 範囲選択
- **Command + A**: すべて選択

### アップロードに失敗する場合
1. ファイルサイズが大きすぎないか確認（100MB以下）
2. ファイル名に特殊文字が含まれていないか確認
3. ブラウザを変えてみる（Chrome推奨）
4. 数個ずつアップロードしてみる
