# 🚀 GitHub公開手順 - 超簡単版

**⏱️ 所要時間: 10分**

---

## 📋 準備するもの
- GitHubアカウント（無料）
- プロジェクトファイル一式

---

## ステップ1: GitHubにログイン（1分）

1. https://github.com にアクセス
2. 「Sign in」でログイン（アカウントがない場合は「Sign up」で無料登録）

---

## ステップ2: リポジトリを作成（2分）

1. 右上の **「+」** → **「New repository」**
2. 以下を入力：
   ```
   Repository name: hitokiwa-lesson-system
   ○ Public （誰でもアクセス可能）
   ```
3. **「Create repository」** をクリック

---

## ステップ3: ファイルをアップロード（3分）

1. **「uploading an existing file」** をクリック
2. 以下のファイルをドラッグ&ドロップ：
   - `index.html`
   - `README.md`
   - `LICENSE`
   - `.gitignore`
   - その他すべての `.md` ファイル
   - `google-apps-script.js`
3. **「Commit changes」** をクリック

---

## ステップ4: 公開設定（2分）

1. **「Settings」** タブをクリック
2. 左サイドバーの **「Pages」** をクリック
3. **Branch** で **「main」** を選択
4. **「Save」** をクリック
5. **1〜5分待つ** ☕

---

## ステップ5: URLを確認（1分）

1. ページをリロード（F5キー）
2. 緑色のボックスに表示されるURLをコピー：
   ```
   https://あなたのユーザー名.github.io/hitokiwa-lesson-system/
   ```

---

## ステップ6: 動作確認（1分）

1. URLにアクセス
2. 「講師」でログイン：
   - ID: `admin`
   - パスワード: `admin123`
3. レッスン一覧が表示されればOK！

---

## 🎉 完了！

公開URLを以下のテンプレートでメンバーに共有してください：

```
📱 hitokiwaレッスン予約システム

URL: https://あなたのユーザー名.github.io/hitokiwa-lesson-system/

【受講生】
ID: 任意（例: STU001）
パスワード: student123

【講師】
ID: admin
パスワード: admin123

【担当者】
ID: coord
パスワード: coord123
```

---

## ❓ うまくいかない場合

### URLにアクセスできない
- 5分待ってリロード
- Settings → Pages で「main」が選択されているか確認

### 404エラーが出る
- URLが正しいか確認
- `index.html` がアップロードされているか確認

### 詳しい手順が知りたい
- 📄 `GITHUB_公開手順.md` を参照
- 📄 `QUICKSTART.md` を参照
