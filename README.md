## ✨ 主要機能

### 🎓 講師機能
- **レッスン作成・管理**: 単一・バッチレッスン作成、編集、削除
- **カレンダー表示**: レッスン詳細（時間・受講者数）付きカレンダー
- **受講生評価**: 受講済み受講生の日本語レベル・態度評価
- **Google Calendar連携**: レッスンの自動カレンダー登録

### 👨‍🎓 受講生機能  
- **レッスン予約**: 全レベルのレッスンに自由に参加可能（対象レベルは目安）
- **📋 予約管理**: 予約済みレッスン・完了済みレッスンの分離表示
- **🤝 同席依頼**: レッスン一覧からコーディネーター同席依頼機能
- **📊 同席履歴**: 過去レッスンの同席参加履歴確認機能
- **講師評価**: 5段階星評価・コメント投稿

### 👥 受講生担当機能
- **🔐 担当者別ログイン**: 各担当者は自分の担当受講生のみを管理
- **👤 テストアカウント**: 'coord'アカウントは全受講生を表示（テスト・管理用）
- **受講生管理**: 受講生情報の登録・編集（担当者割り当て含む）
- **👥 担当受講生一覧**: 自分の担当する受講生のみ表示（未受講・受講済み分離）
- **📋 担当レッスン一覧**: 担当受講生が予約しているレッスンのみ表示
- **📅 講師形式カレンダー**: レッスン時間と受講者数を表示（例：10:00 受講者1名）
- **📱 レッスン詳細モーダル**: 全レッスン一覧・カレンダーから詳細情報・評価・予約者の確認
- **👥 レッスン同席機能**: カレンダーから直接レッスンに同席参加可能
- **統計情報**: 担当受講生の受講率・平均評価の分析
- **低評価受講生管理**: 担当受講生で受講態度に問題のある受講生の一覧・監視

### 🔧 管理者機能（タブ形式）
#### 📊 全評価一覧タブ
- システム全体の評価データ管理
- 低評価監視（講師・受講生）
- 受講生から講師への評価一覧
- 講師から受講生への評価一覧

#### 📈 システム概要タブ
- 講師数・受講生数・レッスン数の統計
- 受講率の総合分析
- 平均受講態度の表示
- データ整合性チェック・クリーンアップ

#### 👥 受講生管理タブ
- 受講生一覧表示（統計情報・担当者情報付き）
- 受講生情報の編集（名前・レベル・担当者）
- 受講生削除（関連データも一括削除）

#### 👨‍💼 担当者管理タブ
- 担当者の追加・編集・削除
- 担当受講生数・受講率の確認
- テストアカウントの保護（削除不可）
- 担当者削除時の受講生再割り当て処理

#### 📊 Google Sheets連携（NEW!）
- **受講生データの自動取得**: スプレッドシートから「求職中」の受講生を自動同期（完全上書き更新）
- **出席・評価データの書き込み**: レッスンの出席状況と評価をスプレッドシートに送信（既存データ保護）
- **ステータス変更者の削除**: 「求職中」でなくなった受講生を確認ポップアップで削除
- **担当者情報の自動抽出**: B列から担当者IDを自動抽出（例：「クオン　CC」→「CC」）
- **担当者未登録時の保護**: 担当者IDを記録し、後から担当者登録時に自動割り当て
- **重複時の完全上書き**: 同じIDの受講生はスプレッドシートの情報で完全更新
- **レベル任意登録**: 日本語レベルは未設定でも登録可能
- **3行目以降への書き込み**: 既存データを保護し、空行のみに書き込み

## 📈 評価システム

### 受講生による講師評価
- **5段階星評価** （★★★★★形式で視覚的表示）
- **小数点精度表示** （例：2.5★★☆☆☆）
- 自由記述コメント
- 評価履歴の保存
- **低評価講師の自動抽出**（☆2以下）

### 講師による受講生評価  
- 日本語レベル評価 (N1-N5)
- **5段階受講態度評価** （色分け表示付き）
  - 非常に良い (5点) - 🟢 緑
  - 良い (4点) - 🔵 青  
  - 普通 (3点) - 🟡 黄
  - 改善が必要 (2点) - 🟠 オレンジ
  - 悪い (1点) - 🔴 赤
- 備考・コメント記録
- 評価済み状態の明確表示
- **問題のある受講生の自動抽出** (2点以下)

### 📊 統計・監視機能
- **受講率分析**: 1回でもレッスンを受けた受講生の割合
- **高度な低評価監視**: 
  - 受講態度問題（改善が必要・悪い）
  - **星評価2以下の受講生** （視覚的に強調表示）
  - 複合的問題受講生の統合表示
- **視覚的統計表示**: 星評価による直感的な評価表示
- **リアルタイム統計**: 各種データの即座更新

## 🔐 データ管理

### 基本データ管理
- **永続化**: LocalStorageによる自動保存
- **データ形式**: JSON形式での構造化データ
- **バックアップ**: ブラウザのデータエクスポート機能対応

### 🧹 データ整合性管理
- **自動クリーンアップ**: システム起動時に自動で古いデータをクリーンアップ
- **レッスン削除時の関連データ削除**: 
  - 予約データ (bookings)
  - 受講生から講師への評価 (ratings)
  - 講師から受講生への評価 (instructorRatings)
- **担当者削除時の処理**:
  - 担当受講生の担当者IDをnullに設定（受講生データは保持）
  - テストアカウント (coord) は削除不可
- **手動クリーンアップ**: 管理者画面で手動実行可能
- **リアルタイム統計更新**: データ変更時の統計情報自動更新

## 📊 データモデル

### LocalStorageキー構造

| キー | 説明 | データ型 |
|------|------|---------|
| `instructors` | 講師情報 | Array<Instructor> |
| `students` | 受講生情報 | Array<Student> |
| `coordinators` | 担当者情報 | Array<Coordinator> |
| `lessons` | レッスン情報 | Array<Lesson> |
| `bookings` | 予約情報 | Array<Booking> |
| `ratings` | 受講生から講師への評価 | Array<Rating> |
| `instructorRatings` | 講師から受講生への評価 | Array<InstructorRating> |

### データ構造

#### Coordinator（担当者）
```javascript
{
  id: string,              // 担当者ID（例: "coord", "coord001"）
  name: string,            // 担当者名
  password: string,        // パスワード
  isTestAccount: boolean,  // テストアカウントフラグ（全受講生表示用）
  createdAt: string        // 作成日時（ISO 8601形式）
}
```

#### Student（受講生）
```javascript
{
  id: string,              // 受講生ID（例: "STU001"）
  name: string,            // 名前（アルファベット大文字）
  level: string,           // 日本語レベル（N1-N5）
  coordinatorId: string,   // 担当者ID（nullの場合は未割り当て）
  createdAt: string        // 作成日時（ISO 8601形式）
}
```

#### Instructor（講師）
```javascript
{
  id: string,              // 講師ID（例: "INST001"）
  name: string,            // 講師名
  password: string,        // パスワード
  createdAt: string        // 作成日時（ISO 8601形式）
}
```

#### Lesson（レッスン）
```javascript
{
  id: string,              // レッスンID（UUID）
  instructorId: string,    // 講師ID
  instructorName: string,  // 講師名
  content: string,         // レッスン内容
  date: string,            // 日付（YYYY-MM-DD形式）
  time: string,            // 時刻（HH:MM形式）
  level: string,           // 対象レベル（N1-N5 or "all"）
  capacity: number,        // 定員
  onlineLink: string,      // オンラインリンク（オプション）
  createdAt: string        // 作成日時（ISO 8601形式）
}
```

#### Booking（予約）
```javascript
{
  id: string,              // 予約ID（UUID）
  lessonId: string,        // レッスンID
  studentId: string,       // 受講生ID
  studentName: string,     // 受講生名
  bookedAt: string         // 予約日時（ISO 8601形式）
}
```

#### Rating（受講生から講師への評価）
```javascript
{
  id: string,              // 評価ID（UUID）
  lessonId: string,        // レッスンID
  studentId: string,       // 受講生ID
  rating: number,          // 評価（0-5、0=不参加）
  comment: string,         // コメント（オプション）
  ratedAt: string          // 評価日時（ISO 8601形式）
}
```

#### InstructorRating（講師から受講生への評価）
```javascript
{
  id: string,              // 評価ID（UUID）
  lessonId: string,        // レッスンID
  studentId: string,       // 受講生ID
  level: string,           // 日本語レベル評価（N1-N5）
  attitude: string,        // 受講態度（"優秀", "良好", "普通", "改善が必要", "悪い", "不参加"）
  comment: string,         // コメント（オプション）
  ratedAt: string          // 評価日時（ISO 8601形式）
}
```
