# 🚀 クイックスタートガイド - テスト運用編

**所要時間: 10分**

このガイドでは、最短でGitHub Pagesにシステムを公開し、テスト運用を開始する手順を説明します。

---

## ステップ1: GitHubにアップロード（3分）

### A. GitHubアカウントでログイン
👉 [GitHub](https://github.com/) にアクセスしてログイン

### B. 新しいリポジトリを作成
1. 右上の **「+」** → **「New repository」**
2. 以下を入力：
   ```
   Repository name: hitokiwa-lesson-system
   Description: hitokiwaレッスン予約システム
   □ Public （または Private）
   □ Add a README file （チェックを外す）
   ```
3. **「Create repository」** をクリック

### C. ファイルをアップロード
1. **「uploading an existing file」** をクリック
2. 以下のファイルをドラッグ＆ドロップ：
   - ✅ `index.html`
   - ✅ `README.md`
   - ✅ `LICENSE`
   - ✅ `.gitignore`
   - ✅ その他すべての `.md` ファイル
3. **「Commit changes」** をクリック

✅ **完了！** ファイルがアップロードされました。

---

## ステップ2: GitHub Pagesを有効化（2分）

### A. Settings を開く
1. リポジトリページで **「Settings」** タブをクリック
2. 左サイドバーの **「Pages」** をクリック

### B. 公開設定
1. **Source** セクション：
   - Branch: **`main`** を選択
   - Folder: **`/ (root)`** を選択
2. **「Save」** ボタンをクリック

### C. 公開URLを確認
1. **1-5分待つ** ☕
2. ページをリロード
3. 緑色のボックスに表示されるURLをコピー：
   ```
   https://あなたのユーザー名.github.io/hitokiwa-lesson-system/
   ```

✅ **完了！** システムが公開されました。

---

## ステップ3: 動作確認（2分）

### A. PCでアクセス
1. 公開URLにアクセス
2. **「講師」** でログイン：
   - ID: `admin`
   - パスワード: `admin123`
3. レッスン作成機能を確認

### B. スマホでアクセス
1. 同じURLにスマホでアクセス
2. **「受講生」** でログイン：
   - ID: `STU001`（任意のID）
   - パスワード: `student123`
3. レッスン一覧を確認

✅ **完了！** システムが正常に動作しています。

## 📱 スマホでの使い方（受講生向け）

### 1. ログイン
1. URLにアクセス
2. 「受講生」ボタンをタップ
3. 任意のIDとパスワードを入力
4. 「ログイン」をタップ

### 2. レッスンを予約
1. 「レッスン一覧」タブで見たいレッスンを探す
2. 「予約する」ボタンをタップ
3. 確認メッセージが表示される

### 3. 講師を評価
1. 「評価」タブに移動
2. 受講したレッスンを探す
3. 「講師を評価する」ボタンをタップ
4. 星の数とコメントを入力
5. 「評価を送信」をタップ

### 4. 同席依頼（オプション）
1. 「レッスン一覧」でレッスンを探す
2. 「同席依頼」ボタンをタップ
3. 確認メッセージが表示される

## 💻 PCでの使い方（講師向け）

### 1. ログイン
1. URLにアクセス
2. 「講師」ボタンをクリック
3. ID: `admin`、パスワード: `admin123`
4. 「ログイン」をクリック

### 2. レッスンを作成
1. 「レッスン作成」タブを開く
2. 必要事項を入力：
   - レッスン内容
   - 日時
   - レベル（デフォルト: 全レベル）
   - 定員
3. 「レッスンを作成」をクリック

### 3. バッチレッスン作成
1. 「レッスン作成」タブで「バッチ作成」を選択
2. 開始日・終了日を設定
3. 複数の時間帯を選択
4. 「バッチ作成を実行」をクリック

### 4. 受講生を評価
1. 「評価」タブに移動
2. 受講済み受講生を選択
3. 日本語レベルと受講態度を評価
4. 備考を入力
5. 「評価を送信」をクリック

## ⚠️ 重要な注意点

### データの保存について
- **各ブラウザに個別保存**: データはユーザー間で共有されません
- **キャッシュクリアで消失**: ブラウザのキャッシュをクリアするとデータが消えます
- **テスト用途のみ**: 本格運用にはバックエンドサーバーが必要

### テスト期間中のルール
1. ✅ テストデータのみ使用
2. ✅ 実名ではなくニックネームを使用
3. ✅ 個人情報は入力しない
4. ✅ 定期的にバックアップ

### データのリセット方法
すべてのデータをクリアして最初からやり直す：
1. ブラウザで **F12** キーを押す
2. 「Console」タブを開く
3. 以下を入力して Enter:
   ```javascript
   localStorage.clear()
   location.reload()
   ```
